## PowerSight-AI
Energy Consumption Prediction using Cloud
📖 Project Overview

This project focuses on predicting future energy consumption using historical data and machine learning techniques. By leveraging time-series forecasting and cloud-based AI services, the model helps in estimating future energy demand accurately.

The dataset used for this project is sourced from Kaggle:
🔗 https://www.kaggle.com/datasets/robikscube/hourly-energy-consumption

🎯 Objective
To analyze historical energy consumption data
To build a predictive model using machine learning
To forecast future energy demand
To deploy the model using cloud platforms for scalability
